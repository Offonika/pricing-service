# S0.1 — Инициализировать репозиторий проекта

## Цель
Создать базовый skeleton репозитория, понятный для всех агентов, с минимальной документацией.

## Что нужно сделать
- Инициализировать Git-репозиторий (если ещё не сделано).
- Добавить `.gitignore` для Python, venv, IDE, Docker.
- Создать минимальный `README.md` с описанием проекта и ссылками на ключевые документы (`docs/PRD.md`, `docs/architecture.md`, `docs/plan.md`).

## Артефакты/файлы
- Корень репозитория: `.gitignore`, `README.md`.

## Definition of Done
- Репозиторий инициализирован, `.gitignore` создан.
- `README.md` содержит краткое описание проекта и ссылки на основные документы.
- Нет лишних/временных файлов в репозитории.
