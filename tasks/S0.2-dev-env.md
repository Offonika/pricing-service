# S0.2 — Настроить базовое окружение разработки

## Цель
Подготовить минимальный набор зависимостей и конфигов, чтобы разработчики могли запустить проект локально.

## Что нужно сделать
- Создать `pyproject.toml` или `requirements.txt` с зависимостями: FastAPI, Uvicorn, SQLAlchemy, Alembic, Pydantic, pytest.
- Добавить конфиги линтеров/форматтеров (`ruff`/`flake8`, `black` или эквивалент).
- Подготовить базовый скрипт/инструкцию запуска приложения в dev-режиме.

## Артефакты/файлы
- `pyproject.toml` или `requirements.txt`
- Конфиги линтеров/форматтеров (`ruff.toml`/`.flake8`, `pyproject` секции, `pyproject` для black и т.п.)
- При необходимости — `scripts/` с утилитами для локального запуска.

## Definition of Done
- Зависимости задекларированы, сборка устанавливается без ошибок.
- Запуск приложения в dev-режиме описан (или есть скрипт).
- Линтер/форматтер могут быть запущены на кодовой базе.
