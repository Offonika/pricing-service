# S0.3 — Настроить Docker для dev-среды

## Цель
Дать воспроизводимое окружение (app + PostgreSQL + Redis) через docker-compose.

## Что нужно сделать
- Создать `infra/docker-compose.yml` с сервисами: `app`, `postgres`, `redis`.
- Добавить `Dockerfile` для приложения (базовый образ Python 3.12+, установка зависимостей).
- Подготовить `.env.example` с переменными окружения для локального запуска (БД, Redis, секреты в виде плейсхолдеров).
- Обновить `README.md` краткой инструкцией `docker-compose up` и миграциями (если уже есть).

## Артефакты/файлы
- `infra/docker-compose.yml`
- `Dockerfile`
- `.env.example`
- Обновления в `README.md`

## Definition of Done
- `docker-compose up` поднимает приложение, БД и Redis без ручных правок файлов.
- Описаны используемые переменные окружения.
- Контейнеры стартуют и видят друг друга по сетке Compose.
